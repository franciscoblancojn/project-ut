# Dependencias

- Node 18.20.2
- PHP 8.3.6

# Ejecutar proyecto

- Clonar repositorio
- Instalar dependencias con npm, yarn, pnpm o bun
```bash
npm i
yarm install
pnpm install
bun i
```
- Instalar dependecias de composer
```bash
composer install --no-autoloader --no-scripts
```
- Crear archivo .env y configurarlo 
- Correr Proyecto
```bash
npm run start
```
